## Student_Management_System
Student Management System is a Backend project using ASP.NET Core MVC. Student details, exams, subject and marks can be added. A dashboard is created for the student and shows grades based on criteria.

## Key Features:
• Built a full-stack ASP.NET Core MVC application with Entity Framework Core, implementing complete CRUD modules for Students, Subjects, Exams, and Marks with relational SQL Server backend.

• Developed a dynamic result generation module using ViewModels, LINQ, and business logic to compute totals, percentages, and grades, with a clean Bootstrap-based UI for result reporting

• Implemented relational data handling with dropdown-driven mark entry, validation, navigation properties, and optimized data access using EF Core queries and include strategies. Technologies Used: C#, ASP.NET Core MVC, Entity Framework Core, SQL Server, Bootstrap
